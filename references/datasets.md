# Dataset Exploration – Knee Osteoarthritis

Final Dataset Selected (Phase 2)

    Name: Knee Osteoarthritis Dataset with Severity Grading
    
    Source: Kaggle (author: shashwatwork / Shashwat Tiwari)
    
    Labeling: Kellgren–Lawrence (KL) grades 0–4 encoded as folder classes
    
    Splits: Predefined train/val/test directories
    
    Why selected: Public access (no NDA), KL labels available, standardized folder layout compatible with PyTorch ImageFolder
    
    Known limitation: Strong class imbalance (KL4 underrepresented)
